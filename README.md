CreceYa - Conecta Tutores y Talentos

Propósito
CreceYa es una plataforma web simple diseñada para conectar personas con experiencia (tutores) en áreas como diseño, programación e idiomas con aprendices interesados en mentorías individuales o grupales. El sitio permite explorar perfiles de mentores, leer reseñas, y contactar para agendar sesiones a través de un formulario.

Estructura del Proyecto

index.html: Estructura principal con secciones semánticas (header, nav, main, sections, footer). Incluye navegación, multimedia (imágenes y un iframe de video), formulario de contacto con Formspree y acceso al sistema de login.

styles.css: Estilos externos con Google Fonts (Roboto), backgrounds, Flexbox (cards de tutores), Grid (reseñas) y media queries para responsividad.

login.js (o script dentro del HTML): Lógica para gestionar el inicio de sesión mediante JavaScript, validando credenciales simples del lado del cliente.

Funcionalidades

Navegación: Enlaces simulados para secciones internas.

Mentores: Cards responsivas con Flexbox.

Reseñas: Layout en Grid.

Contacto: Formulario que envía datos a Formspree.

Login: Sistema básico de autenticación con JavaScript que permite acceder mediante credenciales predefinidas (login sin backend).

Responsivo: Diseño adaptado para móviles y tablets mediante media queries.

Tecnologías

HTML5 semántico.

CSS3 (Flexbox, Grid, Media Queries).

JavaScript (validación de credenciales en el login).

Google Fonts.

Formspree para backend simple en el formulario de contacto.
