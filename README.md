# CreceYa - Conecta Tutores y Talentos

## Propósito
CreceYa es una plataforma web simple diseñada para conectar personas con experiencia (tutores) en áreas como diseño, programación e idiomas con aprendices interesados en mentorías individuales o grupales. El sitio permite explorar perfiles de mentores, leer reseñas, y contactar para agendar sesiones a través de un formulario.

## Estructura del Proyecto
- **index.html**: Estructura principal con secciones semánticas (header, nav, main, sections, footer). Incluye navegación, multimedia (imágenes y iframe de video), y formulario de contacto con Formspree.
- **styles.css**: Estilos externos con Google Fonts (Roboto), backgrounds, Flexbox (cards de tutores), Grid (reseñas), y media queries para responsividad.

## Funcionalidades
- **Navegación**: Enlaces simulados para secciones internas.
- **Mentores**: Cards responsivas con Flexbox.
- **Reseñas**: Layout en Grid.
- **Contacto**: Formulario que envía datos a Formspree.
- **Responsivo**: Se adapta a móviles y tablets con media queries.


## Tecnologías
- HTML5 semántico.
- CSS3 (Flexbox, Grid, Media Queries).
- Google Fonts.
- Formspree para backend simple.
